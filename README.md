# Tutor de Análisis Matemático 2

Este agente de IA está diseñado para actuar como tutor académico especializado en **Análisis Matemático 2**. Su objetivo principal es ayudar al estudiante a comprender teoría, resolver ejercicios y practicar contenidos matemáticos de forma progresiva, clara y justificada.

El agente utiliza como fuente teórica principal el archivo **“TID - Análisis Matemático 1.pdf”**, del cual toma definiciones, conceptos y fundamentos matemáticos. Además, emplea el archivo **“UPC Métodos Matemáticos 2 versión final.pdf”** como material de práctica, ejercicios y entrenamiento.

## ¿Qué hace este agente?

- Explica conceptos de Análisis Matemático 2 desde cero.
- Justifica las explicaciones usando fuentes cargadas.
- Resuelve ejercicios paso a paso.
- Propone ejercicios similares para practicar.
- Corrige respuestas del estudiante indicando errores y mejoras.
- Ayuda a preparar exámenes mediante resúmenes, fórmulas clave, ejercicios modelo y simulacros.
- Relaciona la teoría con la práctica mediante problemas de modelización matemática.

## Temas que trabaja

El agente puede ayudar con:

- Relaciones y funciones.
- Dominio e imagen.
- Funciones lineales y cuadráticas.
- Funciones exponenciales, logarítmicas y trigonométricas.
- Límites.
- Continuidad.
- Derivadas.
- Aplicaciones de derivadas.
- Máximos y mínimos.
- Concavidad y puntos de inflexión.
- Integrales indefinidas.
- Integrales definidas.
- Cálculo de áreas.
- Problemas aplicados y de modelización.

## Método de enseñanza

El agente sigue una metodología de tutoría personalizada:

1. Explica el concepto de forma simple.
2. Lo formaliza matemáticamente.
3. Muestra un ejemplo resuelto.
4. Justifica el procedimiento.
5. Advierte errores comunes.
6. Propone ejercicios de práctica.
7. Corrige las respuestas del estudiante.

## Objetivo

El objetivo del agente no es solo dar respuestas finales, sino enseñar el razonamiento matemático detrás de cada procedimiento. Está pensado para estudiantes que necesitan aprender, repasar o practicar Análisis Matemático 2 con apoyo guiado y fundamentado en material académico.
